参数命令方式说明：


   | 参数       | 类型说明                                             |
   |------------+------------------------------------------------------|
   | separator* | 分隔符                                               |
   | dxfcode*   | 组码                                                 |
   | blkname*   | 块名称                                               |
   | blk*       | "块图元/对象                                         |
   | bulge*     | 凸度                                                 |
   | buzz*      | 容差                                                 |
   | curve*     | 曲线                                                 |
   | prp*       | 块特性                                               |
   | expr*      | "表达式，一般指lisp表达式                            |
   | val*       | 值                                                   |
   | offset*    | 偏移量                                               |
   | path*      | 文件路径                                             |
   | ang*       | "角度值                                              |
   | scale*     | 比例值                                               |
   | segment*   | 线段，两点组成的表，可以表示有向图或无向图的一个边。 |
   | dxf*       | 组码号                                               |
   | ents*      | 图元列表                                             |
   | ent*       | 单个图元                                             |
   | en*        | 单个图元                                             |
   | file*      | 文件名                                               |
   | fp*        | 文件指针                                             |
   | index*     | "索引值                                              |
   | item       | 项或项值                                             |
   | key*       | "键，关键字                                          |
   | label*     | 标签                                                 |
   | desc*      | 描述，简述                                           |
   | obj*       | activeX 对象                                         |
   | ss*        | 选择集                                               |
   | sel*       | 选择集                                               |
   | lst*       | "列表                                                |
   | times*     | "整数，常代指次数                                    |
   | pts*       | "多个坐标点列表                                      |
   | pt*        | 单个2D/3D坐标点                                      |
   | str*       | "字符串                                              |
   | int*       | 整数                                                 |
   | real*      | 实数                                                 |
   | fp*        | 文件流或文件指针                                     |
   | fuzz*      | 容差                                                 |
   | msg*       | "提示信息                                            |
   | filter*    | 过滤dxf组码                                          |
   | params*    | "参数列表                                            |
   | n          | "整数                                                |
